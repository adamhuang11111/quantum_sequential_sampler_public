# analysis code

## Description
```
The codes for analysis are in in the jupyter (ipython) notebook.

├── mean plots - mean prediction plots
├── violin plots - violin plots
├── individual analysis - kernel density plots showing participants' probability judgment distribution, for participants whose quantum interference effect is significant
├── binary complementarity - analysis for binary complementarity
├── Stds - Plots showing the distribution of standard deviation of the final state of the Markov process
├── quantum participants - files containing list of participants whose quantum interference effect is significant by different levels of significance
```

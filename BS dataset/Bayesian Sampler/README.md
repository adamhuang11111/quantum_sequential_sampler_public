# Bayesian Sampler

## Description
```
├── exp1/exp2 names are the pair names in Zhu et al. (2020)
├── beta_sampler: compute the beta likelihood
├── norm_beta: normalize the beta likelihood
├── FitIndBSBeta3: main program for Bayesian sampler
├── MainFitIndBS3: main program for fitting
├── Outerloop3: settings for the bounds and particle swarm algorithm
├── BuildFitIndBS5: build Bayesian sampler script
├── Ms - Mean Prediction
├── nLLS - G square
├── ParmS - best fitted parameters
```

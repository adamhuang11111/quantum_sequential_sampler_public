# Quantum Sequential Sampler

## Description
```
├── exp1/exp2 names are pair names in Zhu et al. (2020)
├── drift_add: Markov transition matrix with additive bias
├── FitIndMarkov3: main program for QSS
├── FitIndMarkov3_classical: main program for CSS
├── BuildFitIndMarkov5: build QSS script
├── FitIndBSBeta3: main program for Bayesian sampler
├── MainFitIndBS3: main program for fitting
├── Outerloop3: settings for the bounds and particle swarm algorithm
```
